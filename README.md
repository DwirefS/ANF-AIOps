### Deployment Process

#### Phase 1: Environment Setup (30 minutes)
```bash
# 1. Clone repository and setup
git clone https://github.com/your-org/anf-aiops.git
cd anf-aiops

# 2. Register Azure resource providers
az provider register --namespace Microsoft.NetApp
az provider register --namespace Microsoft.ContainerInstance
az provider register --namespace Microsoft.Web

# 3. Verify prerequisites
./scripts/verify-prerequisites.sh
```

#### Phase 2: Teams App Registration (15 minutes)
```bash
# 1. Create Azure AD app registration
az ad app create --display-name "ANF AI-Ops Bot" \
  --sign-in-audience AzureADMultipleOrgs

# 2. Configure API permissions
az ad app permission add --id YOUR_APP_ID \
  --api 00000003-0000-0000-c000-000000000000 \
  --api-permissions e1fe6dd8-ba31-4d61-89e7-88639da4683d=Scope

# 3. Create app secret
az ad app credential reset --id YOUR_APP_ID
```

#### Phase 3: Infrastructure Deployment (45 minutes)
```bash
# 1. Configure environment parameters
cd src/infrastructure/bicep
cp environments/dev.parameters.example.json environments/dev.parameters.json
# Edit with your specific values

# 2. Validate deployment
az deployment sub validate \
  --location eastus \
  --template-file main.bicep \
  --parameters @environments/dev.parameters.json

# 3. Deploy infrastructure
./deploy.sh -e dev -l eastus -s YOUR_SUBSCRIPTION_ID

# 4. Verify deployment
az resource list --resource-group anf-aiops-dev-rg --output table
```

#### Phase 4: Application Deployment (30 minutes)
```bash
# 1. Build and deploy MCP Server
cd src/mcp-server
npm install && npm run build
docker build -t anf-mcp-server:latest .
az acr login --name YOUR_ACR_NAME
docker tag anf-mcp-server:latest YOUR_ACR_NAME.azurecr.io/mcp-server:latest
docker push YOUR_ACR_NAME.azurecr.io/mcp-server:latest

# 2. Deploy to Container Apps
az containerapp update \
  --name anf-aiops-dev-mcp \
  --resource-group anf-aiops-dev-rg \
  --image YOUR_ACR_NAME.azurecr.io/mcp-server:latest

# 3. Build and deploy Teams Bot
cd ../teams-bot
npm install && npm run build
zip -r teams-bot.zip dist/ node_modules/ package.json
az webapp deployment source config-zip \
  --resource-group anf-aiops-dev-rg \
  --name anf-aiops-dev-bot \
  --src teams-bot.zip
```

#### Phase 5: Configuration and Testing (20 minutes)
```bash
# 1. Configure application settings
az webapp config appsettings set \
  --resource-group anf-aiops-dev-rg \
  --name anf-aiops-dev-bot \
  --settings @appsettings.json

# 2. Test health endpoints
curl https://anf-aiops-dev-mcp.REGION.azurecontainerapps.io/health
curl https://anf-aiops-dev-bot.azurewebsites.net/health

# 3. Install Teams app and test basic commands
# Upload app manifest to Teams admin center
# Test: /anf help, /anf status
```
